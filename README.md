# 🛍️ ShopX - Modern E-Commerce Frontend

A fully responsive e-commerce application built with **React.js**. This project features real-time data fetching, a persistent cart system, and dynamic product filtering.

## ✨ Key Features
* **Global State Management:** Uses React Context API to manage the cart across all pages.
* **Dynamic Catalog:** Fetches data from DummyJSON API with search and category filters.
* **Responsive Design:** Optimized for mobile and desktop using CSS Grid.
* **Seamless Navigation:** Implemented with React Router v6.

## 🛠️ Tech Stack
* **Frontend:** React, Vite
* **Routing:** React Router DOM
* **Styling:** CSS3 (Flexbox/Grid)
